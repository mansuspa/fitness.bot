# 🏋️ FITNESS BOT

Telegram бот для фитнес тренера с модулями питания, базы данных и обработки команд.

## Возможности

- 📊 **База данных** — хранение информации о пользователях и тренировках
- 🍽️ **Модуль питания** — расчёт калорий и составление рациона
- 💬 **Обработка команд** — удобный интерфейс для взаимодействия
- 🎯 **Telegram Bot API** — интеграция с Telegram

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `.env.example` в `.env`
2. Заполните переменные окружения
3. Запустите бота: `python main.py`
