# 🏋️ FITNESS BOT

Telegram бот для фитнес тренера с планами питания и тренировок для набора и похудения.

## 🎯 Возможности

### 🍽️ Питание
- **Расчёт калорий** по формуле Миффлина-Сан Жеора
- **План питания для НАБОРА** мышечной массы (+300 ккал)
- **План питания для ПОХУДЕНИЯ** (-500 ккал)
- **План питания для ПОДДЕРЖАНИЯ**
- Макронутриенты для каждой цели
- Примеры блюд для каждого приёма пищи

### 💪 Тренировки
- **План тренировок для НАБОРА** (3-4 тренировки в неделю)
- **План тренировок для ПОХУДЕНИЯ** (4-5 тренировок в неделю)
- **План тренировок для ПОДДЕРЖАНИЯ** (3 тренировки в неделю)
- Расписание на неделю
- Упражнения для верхней/нижней части тела
- Кардио и растяжка

### 📊 База данных
- Хранение информации о пользователях
- Профиль пользователя
- Настройки

### 💬 Команды
- `/start` - Начать работу
- `/help` - Справка
- `/menu` - Главное меню
- `/food` - Модуль питания
- `/workout` - Модуль тренировок
- `/profile` - Мой профиль
- `/settings` - Настройки

## 🚀 Как запустить

```bash
# Установка зависимостей
pip install -r requirements.txt

# Настройка
cp .env.example .env
# Отредактируй .env и добавь свой BOT_TOKEN

# Запуск
python main.py
```

## 📝 Примеры планов

### План питания для НАБОРА:
- Завтрак: Овсянка с молоком и орехами, яйца, творог, банан
- Обед: Куриное филе, рис, овощной салат, оливковое масло
- Ужин: Рыба, картофель, овощи, сметана
- Перекусы: Йогурт, орехи, фрукты, протеиновый коктейль

### План тренировок для НАБОРА:
- Понедельник: Грудь + Трицепс
- Вторник: Спина + Бицепс
- Среда: Ноги + Плечи
- Четверг: Отдых
- Пятница: Грудь + Спина
- Суббота: Ноги + Плечи
- Воскресенье: Отдых

## 🔧 Настройка

### Переменные окружения (.env):
```bash

DB_FILE=database.json
MEALS_FILE=meals.json
WORKOUTS_FILE=workouts.json
```
