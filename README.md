# 💪 FITNESS AI TRAINER PRO

Профессиональный Telegram-бот для персональных фитнес-тренировок и питания.

## ✨ Возможности

### 🎯 Основной функционал
- **Выбор цели**: похудение или набор массы
- **Тренировочные планы**: недельная программа + упражнения на день
- **План питания**: калории, белки, углеводы, жиры
- **Калькулятор калорий**: расчет TDEE по формуле Миффлина-Сент-Джеора
- **AI-тренер**: ответы на вопросы пользователя
- **Отслеживание прогресса**: логирование тренировок

### 💎 Premium функции
- Персональный план на неделю
- Отслеживание прогресса
- AI-рекомендации
- Быстрый расчет калорий
- Подробная аналитика

## 🚀 Быстрый старт

### Требования
- Python 3.10+
- pip

### Установка

1. **Клонируй репозиторий**
```bash
git clone https://github.com/mansuspa/fitness.bot.git
cd fitness.bot
```

2. **Создай виртуальное окружение**
```bash
python -m venv venv
source venv/bin/activate  # На macOS/Linux
venv\Scripts\activate      # На Windows
```

3. **Установи зависимости**
```bash
pip install -r requirements.txt
```

4. **Создай файл .env**
```bash
cp .env.example .env
```

5. **Добавь токен бота в .env**
```
BOT_TOKEN=your_token_here
```

6. **Запусти бота**
```bash
python main.py
```
