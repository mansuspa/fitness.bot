# 🏋️ FITNESS BOT

Telegram бот для фитнес тренера с планами тренировок и питания для набора и похудения.

## ✨ Возможности

### 🍽️ Модуль питания
- **Расчёт калорий** по формуле Миффлина-Сан Жеора
- **План питания для набора массы** — больше углеводов
- **План питания для похудения** — меньше углеводов
- **Распределение по приёмам пищи:** завтрак, обед, ужин, перекусы

### 💪 Модуль тренировок
- **План тренировок для набора массы:**
  - Верх тела: жим, подтягивания, тяга
  - Низ тела: приседания, становая тяга, выпады
  - Кардио: бег, велосипед, эллипсоид
  - Отдых: 2-3 минуты между подходами

- **План тренировок для похудения:**
  - Верх тела: отжимания, планка, подтягивания
  - Низ тела: выпады, приседания, бёрпи
  - Кардио: интервальный бег, 30-40 минут
  - Отдых: 60-90 секунд между подходами

### 📊 База данных
- Хранение информации о пользователях
- Сохранение параметров (вес, рост, возраст, цель)

### 💬 Обработка команд
- Удобный интерфейс для взаимодействия
- Inline-клавиатуры для навигации

## 🚀 Команды бота

- `/start` - Начать работу
- `/help` - Справка
- `/menu` - Главное меню

### Питание
- `/food` - Расчёт калорий
- `/food_gain` - План питания для набора массы
- `/food_loss` - План питания для похудения

### Тренировки
- `/workout_gain` - План тренировок для набора массы
- `/workout_loss` - План тренировок для похудения

### Профиль
- `/profile` - Мой профиль

## 🛠️ Установка

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/mansuspa/fitness.bot.git
   cd fitness.bot
   ```

2. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Настрой переменные окружения:
   ```bash
   cp .env.example .env
   ```

4. Отредактируй `.env` и добавь свой `BOT_TOKEN`

5. Запусти бота:
   ```bash
   python main.py
   ```
